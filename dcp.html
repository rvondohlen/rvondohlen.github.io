Preserving a Bit of D.C.’s History

Description

An interactive map made to explore and document old-fashioned emergency call boxes

Deliverable

Mobile Optimized Web-App

Role

Design & Development

Technologies

Backbone.js, Leaflet/Mapbox.js, & Firebase

Client

Self-initiated

Visit Site

This photo I posted to Instagram marks the beginning of the project. As you can see in the comments, my friend Lauren so helpfully pointed out that these odd looking totems were call boxes and so the search for more began.

Thanks @laurenbwright

As I found more around the city, I kept a list of their coordinates on Github and took photos of a few. What really drew me in was the way they seemed to embody the feeling of the neighborhoods around them. By documenting them I hoped to preserve their stories as they begin to be removed.

From a simple list, the project grew into an interactive map. At the time, I was looking to grow my javascript skills so adding some functionality to add boxes through the map interface was a good challenge. Backbone.js turned out to be an easy to use library to control both the views and data handling with little overhead.

The following set of views show the full user flow through the process of adding a new marker as well as exploring the info drawer.

Here is a quick video screen capture of the new marker placement user flow.

It was important for me to have the interface feel fast and responsive as a native app would.

This project is still ongoing and that being said, I'll continue to add more boxes and I'm considering way to add in the ability to upload photos as well as improving the overall performance.
