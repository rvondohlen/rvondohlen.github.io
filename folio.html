Visit Site

As, what I would consider, a pretty ordinary internet user, I save a lot of stuff from the web. I've tried many services and tools that allow you to save and organize all that stuff in visual collection, but not many have been able to give me exactly what I want and need. My favorite, GimmeBar - which was perfect, by the way - has since been left unsupported by it's creators and no longer works properly. My main needs are a easy-to-browse, visual layout, complete access to all of the raw files, and an open organizational system that I can define. GimmeBar had a solid visual layout, backed up to Dropbox, and employed a tag based structure that fit the bill perfectly. But alas, it's gone, so I set out to create a more sustainable, self-hosted solution for myself.

Folio is a simple Jekyll theme made for organizing images. Tags are used to dynamically populate navigation and it can be run locally to give immediate access to files and also makes for a pretty fast a fluid experience.

In it's current state, I use it mostly for moodboards and brand research projects, but it could work reasonably well as a small portfolio or gallery.

Taking a look yourself will do it better justice than any images I could add here. Please check out the demo if you haven't already
